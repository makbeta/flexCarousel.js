You can get started with using flexCarousel.js by downloading the required files from the GitHub repository.

The only dependency that flexCarousel.js has, is jQuery which you need to load before the plugin. After you have loaded both jQuery and flexCarousel.js, you need to initialize the plugin. You can also use the slim version of jQuery if preferred.
