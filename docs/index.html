Introduction

flexCarousel.js is a simple, lightweight, responsive jQuery plugin which allows you to create carousels with Flexbox.

Getting Started

You can get started with using flexCarousel.js by downloading the required files from the GitHub repository.

The only dependency that flexCarousel.js has is jQuery which you need to load before the plugin. After you have loaded both jQuery and flexCarousel.js, you need to initialize the plugin.

flexCarousel.js is currently in the early stages of development and only supports 1 slide to be visible at once for the time being. Feel free to submit a pull request on GitHub to help contribute!

This is what your HTML structure should look like:

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="flexCarousel.css">
  </head>
  <body>
    <div class="flexCarousel">
      <div class="flexCarousel-container">
        <div class="flexCarousel-slides">
          <div class="flexCarousel-slide">1</div>
          <div class="flexCarousel-slide">2</div>
          <div class="flexCarousel-slide">3</div>
          <div class="flexCarousel-slide">4</div>
          <div class="flexCarousel-slide">5</div>
        </div>
      </div>
    </div>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script src="flexCarousel.js"></script>
    <script>
    $(document).ready(function() {
      $('.flexCarousel').flexCarousel();
    });
    </script>
  </body>
</html>

Compabitility

flexCarousel.js uses Flexbox features to manage the different carousel slides and overall layout. According to Can I use, this jQuery plugin is compatible with recent versions of the following browsers:

  • Chrome
  • Edge
  • Firefox
  • Opera
  • Safari

Internet Explorer is only partially supported.

Usage

Autoplay

To enable autoplay, there are no changes needed for the HTML markup, just simply enable the option (boolean) when you initialize the plugin.

<script>
$(document).ready(function() {
  $('.flexCarousel').flexCarousel({
    autoplay: true
  });
});
</script>

Different Autoplay Speed

To change how long the active slide will stay active for before transitioning to the next slide, there are no changes needed for the HTML markup, just simply set a value (integer, in milliseconds) to the option when you initialize the plugin. autoplay must be set to true for this option to work.

<script>
$(document).ready(function() {
  $('.flexCarousel').flexCarousel({
    autoplay: true,
    autoplaySpeed: 1000
  });
});
</script>

Different Prev/Next Icons

To change the icons used for the prev/next buttons, there are no changes needed for the HTML markup, just simply set a value (string) to the option when you initialize the plugin. flexCarousel.js supports Font Awesome 5 out of the box, but this can easily be replaced for a different icon set if you prefer.

<script>
$(document).ready(function() {
  $('.flexCarousel').flexCarousel({
    nextArrow: '<i class="fas fa-arrow-circle-right"></i>',
    prevArrow: '<i class="fas fa-arrow-circle-left"></i>'
  });
});
</script>

Fixed Height Carousel

Out of the box, flexCarousel.js slides will only take up as much height as it needs. In the case you need to set a specific height for your carousel, you can use the height option with a value (string).

<script>
$(document).ready(function() {
  $('.flexCarousel').flexCarousel({
    height: '600px',
  });
});
</script>

Image Carousel

To use images for your carousel, all you have to do is add the image inside the child slide element. The img tag should be wrapped in a <figure class="flexCarousel-image"> tag to keep the image responsive at different sizes. Ideally, your images should all be the same height so that there will be no whitespace on other slides where the images are shorter.

In the case that this does happen, all images will be vertically centered.

<div class="flexCarousel">
  <div class="flexCarousel-container">
    <div class="flexCarousel-slides">
      <div class="flexCarousel-slide">
        <figure class="flexCarousel-image">
          <img src="assets/img/puppy1.jpg" alt="Puppy" width="697" height="400">
        </figure>
      </div>
      <div class="flexCarousel-slide">
        <figure class="flexCarousel-image">
          <img src="assets/img/puppy2.jpg" alt="Puppy" width="600" height="400">
        </figure>
      </div>
      <div class="flexCarousel-slide">
        <figure class="flexCarousel-image">
          <img src="assets/img/puppy3.jpg" alt="Puppy" width="692" height="400">
        </figure>
      </div>
      <div class="flexCarousel-slide">
        <figure class="flexCarousel-image">
          <img src="assets/img/puppy4.jpg" alt="Puppy" width="600" height="400">
        </figure>
      </div>
    </div>
  </div>
</div>

Image with Captions Carousel

To add a caption to your image, simply add the figcaption tag below your image in the slide.

<div class="flexCarousel-slide">
  <figure class="flexCarousel-image">
    <img src="assets/img/puppy1.jpg" alt="Puppy" width="697" height="400">
  </figure>
  <figcaption>A cute puppy</figcaption>
</div>

Prev/Next Slide Overlay Icons

By default, the prev/next icons are placed outside of the flexCarousel-slides container. If you prefer the icons to overlay onto the slides, just simply enable the option (boolean) when you initialize the plugin.

<script>
$(document).ready(function() {
  $('.flexCarousel').flexCarousel({
    arrowsOverlay: true,
  });
});
</script>

Video Carousel

To add a video to your flexCarousel.js slide, simply use the HTML5 video tag. If your video has audio and you wish it to autoplay when the user goes to it's respective slide, you must use both the autoplay and muted attributes for it to work in certain browsers.

<div class="flexCarousel-slide">
  <video autoplay muted>
    <source src="assets/vid/bunny1.mp4" type="video/mp4">
  </video>
</div>

Options

  • arrows: (default true). Determines whether a previous and next arrow should be shown on either side of the slides so that the user can change the active slide.
  • arrowsOverlay: (default false). arrows must be set to true for this option to work. Determines whether the arrows overlay onto the active slide or have their own wrapping elements.
  • autoplay: (default false). Determines whether the active slide changes to the next slide on a time delay.
  • autoplaySpeed: (default 5000). autoplay must be set to true for this option to work. Determines how long the active slide will stay active for before transitioning to the next slide.
  • height: (default: null). Defines whether a fixed height is to be added to the .flexCarousel parent element.
  • nextArrow: (default <i class="fas fa-angle-right"></i>). Defines the HTML code for the element which changes the active slide to the next slide when clicked on. Out of the box, flexCarousel.js supports Font Awesome 5.
  • prevArrow: (default <i class="fas fa-angle-left"></i>). Defines the HTML code for the element which changes the active slide to the previous slide when clicked on. Out of the box, flexCarousel.js supports Font Awesome 5.
